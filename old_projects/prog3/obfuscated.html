Obfuscated C code

Il est très facile d'écrire du code difficile à relire. Voici quelques sources en C qui ont été écrit dans le cadre d'un concours où le but est justement d'écrire du code obscure. Il est bien évident que ceci est un contre example de ce que vous devez faire !

  • pi.c - Voici un programme qui est écrit en forme de cercle. Le programme calcule sa propre aire et diamètre et ensuite effectue une division pour afficher pi.
  • queens.c - Un programme très court qui resoud le fameux problème des 8 reines sur un échiquier. Le but consiste à placer 8 reines sur un échiquier de sorte qu'aucune ne soit en prise. On sait que dans le jeu d'échecs, une reine peut se déplacer en ligne droite sur toute ligne, colonne ou diagonale.
